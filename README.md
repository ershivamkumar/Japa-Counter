# Japa Counter Web App 🙏

A **mobile-friendly devotional web application** that helps users count mantra **Japa (chanting)** digitally.
The app simulates a **108-bead mala**, tracks completed malas, and allows users to listen to **devotional meditation chants** while chanting.

---

## ✨ Features

• 📿 **108 Mala Counter** – Visual circular progress for each mala
• 🔢 **Tap to Jap Button** – Tap to increase mantra count
• 📊 **Malas Tracker** – Automatically counts completed malas (108 chants)
• 🔢 **Total Counter** – Tracks total chants done in the session
• 🎵 **Multiple Devotional Songs** – Users can play meditation/chanting audio
• 📳 **Vibration Feedback** – Small vibration on each tap (supported devices)
• 📱 **Mobile Friendly UI** – Designed for smartphone use

---

## 🧘 Devotional Chants Included

The application includes the following mantra audio tracks:

* Om Gauri Shankaraya Namah (108 Times)
* Om Namah Shivaya (108 Times)
* Radhe Radhe (108 Times)
* Shri Ram Jai Ram Jai Jai Ram (108 Times)
